# 🧠 AI Personality Analyzer

An AI-powered web application that predicts personality traits based on daily lifestyle patterns.

## 🚀 Features
- 🎯 Predict Introvert vs Extrovert
- 📊 ML model using Random Forest
- 🤖 AI-generated personalized suggestions
- 🌐 Live deployed web app (Streamlit)
- ⚡ Beautiful animated UI

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- OpenAI API

## 📊 Input Features
- Social Time
- Screen Time
- Sleep Hours
- Study Hours

## 🧠 Model
- Algorithm: Random Forest Classifier
- Accuracy: ~85%

## 🔥 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
